4 As funções hiperbólicas são funções pares ou ímpares?

Nessa seção iremos verificar quais funções hiperbólicas são ímpares ou pares:

4.1 Seno hiperbólico

4.1.1 Definindo \(\sinh(-x)\)

\[\sinh(x) = \dfrac{e^{x} - e^{-x}}{2}\]

\[\sinh(-x) = \dfrac{e^{-x} - e^{x}}{2} \]

4.1.2 Definindo \(-\sinh(x)\)

\[\sinh(x) = \dfrac{e^{x} - e^{-x}}{2}\]

\[-\sinh(x) = - \dfrac{e^{x} - e^{-x}}{2}\]

\[-\sinh(x) = \dfrac{-e^{x} + e^{-x}}{2}\]

4.1.3 \(\sinh(-x)\) é igual a \(-\sinh(x)\)?

\[\sinh(-x) \quad = \quad -\sinh(x) \]

\[ \dfrac{e^{-x} - e^{x}}{2} \quad = \quad \dfrac{-e^{x} + e^{-x}}{2} \]

Sim, o seno hiperbólico é função ímpar.

4.2 Tangente hiperbólica

4.2.1 Definindo \(\tanh(-x)\)

\[\tanh(x) = \dfrac{e^{x} - e^{-x}}{e^{x} + e^{-x}}\]

\[\tanh(-x) = \dfrac{e^{-x} - e^{x}}{e^{-x} + e^{x}}\]

4.2.2 Definindo \(-\tanh(x)\)

\[\tanh(x) = \dfrac{e^{x} - e^{-x}}{e^{x} + e^{-x}}\]

\[-\tanh(x) = -\dfrac{e^{x} - e^{-x}}{e^{x} + e^{-x}}\]

\[-\tanh(x) = \dfrac{-e^{x} + e^{-x}}{e^{x} + e^{-x}}\]

4.2.3 \(\tanh(-x)\) é igual a \(-\tanh(x)\)?

\[\tanh(-x) \quad = \quad -\tanh(x) \]

\[ \dfrac{e^{-x} - e^{x}}{e^{-x} + e^{x}} \quad = \quad \dfrac{-e^{x} + e^{-x}}{e^{x} + e^{-x}} \]

Sim, a tangente hiperbólica é função ímpar.

4.3 Cotangente hiperbólica

4.3.1 Definindo \(\coth(-x)\)

\[\coth(x) = \dfrac{e^{x} + e^{-x}}{e^{x} - e^{-x}} \quad = \quad \dfrac{1}{\tanh(x)}\]
\[\coth(-x) = \dfrac{e^{-x} + e^{x}}{e^{-x} - e^{x}} \]

4.3.2 Definindo \(-\coth(x)\)

\[-\coth(x) = \dfrac{e^{x} + e^{-x}}{e^{x} - e^{-x}} \quad = \quad \dfrac{1}{\tanh(x)}\]

\[-\coth(x) = -\dfrac{e^{x} + e^{-x}}{e^{x} - e^{-x}} \]

\[-\coth(x) = \dfrac{e^{x} + e^{-x}}{-e^{x} + e^{-x}} \]

4.3.3 \(\coth(-x)\) é igual a \(-\coth(x)\)?

\[\coth(-x) \quad = \quad -\coth(x) \]

\[ \dfrac{e^{-x} + e^{x}}{e^{-x} - e^{x}} \quad = \quad \dfrac{e^{x} + e^{-x}}{-e^{x} + e^{-x}} \]

Sim, a cotangente hiperbólica é função ímpar.

4.4 Cossecante hiperbólica

4.4.1 Definindo \(csch(-x)\)

\[csch(x) = \dfrac{2}{e^{x} - e^{-x}} \quad = \quad \dfrac{1}{\sinh(x)}\]

\[csch(-x) = \dfrac{2}{e^{-x} - e^{x}}\]

4.4.2 Definindo \(-csch(x)\)

\[csch(x) = \dfrac{2}{e^{x} - e^{-x}} \quad = \quad \dfrac{1}{\sinh(x)}\]

\[-csch(x) = -\dfrac{2}{e^{x} - e^{-x}}\]

\[-csch(x) = \dfrac{2}{-e^{x} + e^{-x}}\]

4.4.3 \(csch(-x)\) é igual a \(-csch(x)\)?

\[csch(-x) \quad = \quad -csch(x) \]

\[ \dfrac{2}{e^{-x} - e^{x}} \quad = \quad \dfrac{2}{-e^{x} + e^{-x}} \]

Sim, a cossecante hiperbólica é função ímpar.

4.5 Cosseno hiperbólico

4.5.1 Definindo \(\cosh(-x)\)

\[\cosh(x) = \dfrac{e^{x} + e^{-x}}{2}\]

\[\cosh(-x) = \dfrac{e^{-x} + e^{x}}{2}\]

4.5.2 \(\cosh(x)\) é igual \(\cosh(-x)\)?

\[\cosh(x) \quad = \quad -\cosh(x) \]

\[ \dfrac{e^{x} + e^{-x}}{2} \quad = \quad \dfrac{e^{-x} + e^{x}}{2} \]

Sim, o seno hiperbólica é função par.

4.6 Secante hiperbólica

4.6.1 Definindo \(sech(-x)\)

\[sech(x) = \dfrac{2}{e^{x} + e^{-x}} \quad = \quad \dfrac{1}{\cosh(x)}\]

\[sech(-x) = \dfrac{2}{e^{-x} + e^{x}}\]

4.6.2 \(sech(x)\) é igual a \(sech(-x)\)?

\[sech(x) \quad = \quad sech(-x) \]

\[ \dfrac{2}{e^{x} + e^{-x}} \quad = \quad \dfrac{2}{e^{-x} + e^{x}} \]

Sim, a secante hiperbólica é função par.
